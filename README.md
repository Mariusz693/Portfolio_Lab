### Main functionalities:
- registration
- login
- add donations
- contact form (send e-mail).
- list of intitutions broken down into:
    - foundations
    - non-governmental organization
    - local donations
- statistics all donations

### Technologies:
* Django (Python's framework)
* JS (Fetch)
